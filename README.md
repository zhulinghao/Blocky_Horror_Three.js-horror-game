# 方块噩梦 (Blocky Horror)

一款基于 Three.js 的第一人称恐怖游戏。你需要在阴森诡异的豪宅中逃脱，同时躲避神秘的幽灵追击。

## 游戏特性

- 🏚️ **3D 开放世界** - 随机生成的豪宅地图，多个房间和隐藏点
- 👻 **AI 幽灵** - 智能追踪的敌人，会通过听觉感知玩家
- 🔦 **手电筒系统** - 探照灯照明，可切换开关
- 🚪 **交互对象** - 橱柜藏匿、电灯开关、钥匙拾取等
- 🗺️ **小地图** - 实时地图导航，可切换大小
- 🎵 **沉浸式音效** - 动态心跳声效，环境音乐
- ⌨️ **流畅操控** - WASD 移动、鼠标观看、多种交互选项

## 游戏操控

| 按键 | 功能 |
|-----|-----|
| **WASD** | 移动 |
| **Space** | 跳跃 |
| **Shift** | 奔跑 |
| **E** | 交互/躲藏 |
| **F** | 切换手电筒 |
| **M** | 切换小地图大小 |
| **鼠标移动** | 观看（鼠标速度可在菜单调整） |
| **Esc** | 弹出菜单/暂停游戏 |

## 游戏目标

1. 在豪宅中寻找**逃脱门钥匙**
2. 躲避幽灵的追击（使用橱柜等隐藏点）
3. 到达出口门门安全逃脱

## 游戏提示

- 💡 利用电灯开关改变环境亮度，视野会影响幽灵的追踪
- 🤐 尽量保持安静，不要让幽灵听到你
- 📦 在橱柜等地方可以暂时躲避幽灵的追击
- 📡 拾取雷达装置可以在小地图上看到幽灵位置

## 开发技术

- **Three.js** - 3D 图形渲染
- **Vanilla JavaScript** - 游戏逻辑和交互
- **Web Audio API** - 程序化音效生成
- **WebGL** - GPU 加速渲染

## 文件结构

```
src/
├── game.js              # 主游戏控制器和循环
├── player.js            # 玩家控制和摄像头
├── ghost.js             # 幽灵 AI 和行为
├── world.js             # 世界生成和碰撞检测
├── minimap.js           # 小地图渲染
├── sound_generator.js   # 程序化音效生成
├── localization.js      # 本地化文本管理
└── utils.js             # 工具函数
```

## 本地化支持

游戏支持多语言，可在 `localization.js` 中修改语言设置。
